# The Wild Oasis

This project, named **"The Wil Oasis,"** is a web application built using React and various related libraries. It includes features for managing users, bookings, cabin details, and more. The backend is powered by Supabase.

## Features
- **User Management**: The admin can create users who can then access the application.  
- **Booking Management**: Users can check their bookings and manage cabin details.  
- **Customizable Settings**: Admin can update various parameters such as breakfast price, minimum and maximum number of stays, and maximum number of guests.  
- **Dashboard**: The dashboard provides visual representations of booking data, stay duration summary, recent sales, occupancy rate, etc.  
- **Profile Management**: Users can add profile pictures and update their password, email, and name.  

## Technologies Used
- React  
- React Router  
- Context API  
- Hot-toast  
- React-icons  
- React-recharts  
- Styled Components  
- Supabase (Backend)  


## Getting Started
To run this project locally, follow these steps:  
- Clone this repository to your local machine.  
- Install dependencies by running npm install or yarn install.  
- Start the development server with npm start or yarn start.  
- Access the application in your web browser at http://localhost:3000.
